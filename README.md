# Noted

An app that helps you keep track of the things that you care about.

## Status

[![codecov](https://codecov.io/gh/NotedApp/noted_app/branch/master/graph/badge.svg?token=NM7A13VH6K)](https://codecov.io/gh/NotedApp/noted_app)

## Setup

1. Clone this repository to the desired location and open a terminal in the root `noted_app` folder.
2. Run the setup script with `bash ./setup.sh`.
3. Run the `flutter pub get` command.
4. Navigate to the `noted_catalog` subdirectory.
5. Run the `flutter pub get` command.
